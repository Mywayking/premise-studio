# Premise Studio - UI Design System

## 设计哲学

Premise Studio 不是 AI Tool。

而是：
创作空间。

## 视觉关键词

- 余白
- 米纸
- 日式
- 编辑器感
- 低噪音
- 柔和

## 色彩系统

### Background
- #F6F1E8
- #F8F5EF
- #EFE7DA

### Text
- #1F1F1F
- #3A3A3A
- #6B6B6B

### Accent
- #B0433A

## Layout

### Desktop
左：创作树
中：当前创作
右：AI 陪练

### Mobile
单栏模式

## Card Design

Card 必须：
- 大圆角
- 大 padding
- 呼吸感
- 大量留白

禁止：
- AI 聊天气泡
- 深色代码块风格
