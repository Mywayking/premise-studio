# Premise Studio - V1 Product PRD

## 产品定位

Premise Studio 是一个将《手把手教你玩脱口秀》的创作方法论工程化、节点化、工作流化的 AI 单口喜剧创作工作台。

不是：
- AI 段子生成器
- AI 聊天机器人
- 一键生成稿子工具

而是：
- 单口喜剧创作操作系统（Comedy IDE）

## V1 核心目标

让用户完成：

素材
→ 前提
→ 角度
→ 草稿
→ 改稿

完整创作闭环。

## V1 边界

### 支持
- 文本输入
- 长文本输入
- 聊天记录输入
- rant 输入
- 完整稿子输入

### 不支持
- 图片输入
- 视频输入
- 语音输入
- 演出分析
- 社区
- 协作

## 核心哲学

AI 不负责替用户写。

AI 负责：
- 提炼
- 拆解
- 推演
- 激发
- 陪练
- 改稿
