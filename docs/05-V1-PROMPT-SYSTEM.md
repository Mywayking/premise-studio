# Premise Studio - Prompt System

## Prompt 原则

禁止：
“帮我写一个脱口秀段子”

必须：
分阶段创作。

## Premise Prompt

任务：
- 提炼观点
- 提炼矛盾
- 提炼世界观

禁止：
- 直接生成完整段子

## Angle Prompt

任务：
探索不同角度：
- 字面化
- 系统化
- 情绪化
- 身份错位

## Draft Prompt

任务：
生成可表演草稿。

必须：
- 像真人说话
- 有停顿感
- 有情绪
- 不像作文

## Rewrite Prompt

任务：
- 增强 punch
- 增强节奏
- callback
- 更短
- 更狠
